Proyecto: Organizador de Eventos – Android (Kotlin)

Aplicación móvil nativa desarrollada en Android Studio con Kotlin, cuyo objetivo es gestionar eventos personales como citas, juntas, entregas de proyecto, exámenes y más.
Este proyecto forma parte del desarrollo de una aplicación completa que incluye registro, consulta, modificación, notificaciones y respaldo en la nube.

Funcionalidades principales (planeadas)
Menú hamburguesa con acceso a:
- Añadir eventos
- Consultar / modificar eventos
- Respaldo en Dropbox o Google Drive
- Restauración de datos
- Acerca de…
- Salir

Bottom Navigation Bar con:
- Inicio
- Consultar
- Salir

Gestión de eventos, incluyendo:
- Selección de contacto desde la agenda del usuario
- Selección de ubicación desde mapa
- Fecha, hora, tipo de evento, estatus
- Recordatorios personalizados (día/hora, 10 min antes, 1 día antes, o sin notificación)

Consultas avanzadas:
- Por fecha
- Por rango de fechas
- Por mes
- Por año
- Por tipo de evento
- Respaldo y restauración en Google Drive o Dropbox
- Notificaciones programadas

Tecnologías previstas
- Kotlin
- Android Studio
- Room / SQLite
- Google Maps / Places API
- AlarmManager / WorkManager
- Drive API / Dropbox API

Estado del proyecto

En proceso de estructura inicial, diseño de pantallas y configuración del proyecto.
